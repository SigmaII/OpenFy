# OpenFy
music player app (for linux)
# Installation
- ``` pip3 install -r requirements.txt ```
# Usage
| it's recommended create a virtual environment for this app |
- ``` python -m venv $HOME/envFy ```
- ``` source envFy/bin/activate ```
- ``` cd Openfy ```
- ``` python3 app.py ```
